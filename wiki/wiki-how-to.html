Wiki Structure

The Wiki is split into 2 sections:

  1. Website
    1. Contains all files related to the River City Rainbow Collective website.
    2. This is where our resource directory and blog resides. 
    3. Access to the website files is restricted to the Website Editor role. 
    4. Files created in here must be made using the Markdown editor.
    5. No delete access
  2. Wiki
    1. General use wiki
    2. Everyone with Discord access can create and edit anything in the /wiki directory
    3. No delete access

The goal of the wiki overall is to consolidate information that we want to store and organize for later use. The wiki directory can become a central pool of information that can be copied into the website directory to be included on our website.

Feel free to mess around (on your own file) and contribute any information you feel is important to document.
