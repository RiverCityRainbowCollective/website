Welcome to the Wiki!

Here is where we'll put all our important instructions on how to contribute to the RC2 Resource Website

Contributing to the Website

If you look at the sidebar on the left, you'll see a website folder. This folder contains all the markdown (.md) files that will end up on the website. Everything outside of that folder will not be included.

Website File Structure

The structure of the files inside the website folder are hierarchical by category. This hierarchy will end up as the base navigation inside the website. You can create the structure as needed to add additional categories of resources. You accomplish this by editing the file path when you create a new page.

Only Markdown files are supported on the website!

All files inside the website folder should be created via the Markdown text editor. The website software we use accepts markdown files and renders them into HTML files before the website is fully deployed.

If you create a file with a different editor (not markdown) inside the website folder, it won't break anything but that page will not show up in the website. There is an option to “convert” the page to another format, you can try that to convert it to markdown, but there's a risk of loss of formatting, so you may need to revisit the format of the page after conversion.
